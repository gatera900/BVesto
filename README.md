## Getting Started

1. **Clone the repository:**

   ```
   git clone https://github.com/gatera900/BVesto.git
   cd BVesto
   ```

2. **Install dependencies:**

   ```
   npm install
   ```

3. **Run the development server:**

   ```
   npm run dev
   ```

4. **Open your browser:**  
   Visit [http://localhost:5173](http://localhost:5173) to view the app.
